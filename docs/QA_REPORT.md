## 5. حدود معروفة / ليست جاهزة للإطلاق بعد (تحتاج إجراءً)

1. **نشر Vercel + اختبار حيّ:** البيئة المعزولة هون تحجب الوصول لـ Supabase (سياسة شبكة 403)، فما قدرنا نعمل اختبار تفاعلي حيّ كامل (تسجيل→دخول→مزامنة). **مطلوب:** نشر على Vercel (env vars) + اختبار دخول فعلي على الرابط.
2. **روابط رجوع Supabase Auth:** لازم إضافة رابط Vercel في Authentication → URL Configuration (Site URL + Redirect URLs) وإلا تأكيد البريد/الاستعادة ما يشتغل.
3. **محتوى بنك الأسئلة الرسمي:** حالياً بيانات seed تجريبية. لما توصل ملفات صاحب المشروع، شغّل `scripts/import-questions.ts` (يُعدَّل `mapRow` حسب بنية الملفات).
4. **الشعار:** عنصر نائب بهوية design.md — يُستبدل بالأصلي بـ`public/brand/` (نفس المسارات) + إعادة توليد الأيقونات.
5. **Web Push مغلق التطبيق:** حالياً إشعارات محلية أثناء الفتح. Push حقيقي يحتاج مفاتيح VAPID + مسار اشتراك/إرسال فوق SW الجاهز.
6. **reward_events:** العرض مشتقّ من الإتقان الفعلي؛ كتابة الأحداث للجدول مؤجّلة (تُستخدم مستقبلاً للسلاسل).
